# Architecture Overview

## System Architecture

NeuroVault is a Tauri v2 desktop application with a Rust backend, React frontend, and SQLite database. All processing runs locally -- no cloud services required.

```
+-------------------+       +-------------------+       +-------------------+
|    React 19       |       |    Tauri v2        |       |    Ollama         |
|    Three.js       | <---> |    Rust Backend    | <---> |    Local LLM      |
|    Zustand        |  IPC  |    (Tokio async)   | HTTP  |    Embeddings     |
+-------------------+       +-------------------+       +-------------------+
                                    |
                            +-------+-------+
                            |               |
                    +-------+---+   +-------+---+
                    |  SQLite   |   |   HNSW    |
                    |  FTS5     |   |  Vector   |
                    |  WAL mode |   |  Index    |
                    +-----------+   +-----------+
```

### Layer Breakdown

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Desktop Shell | Tauri v2 | Native window, system tray, IPC bridge |
| Frontend | React 19 + Three.js + Zustand | 3D visualization, UI, state management |
| Backend | Rust (Tokio async runtime) | All business logic, circuits, API |
| Database | SQLite + FTS5 (bundled) | Knowledge storage, full-text search |
| Vector Search | HNSW (instant-distance, pure Rust) | Semantic similarity search |
| LLM / Embeddings | Ollama (local) or Anthropic API | AI-powered analysis and embeddings |
| MCP Bridge | Node.js stdio server | AI assistant integration |
| HTTP API | Axum (localhost:17777) | External programmatic access |

### Data Directory Layout

```
~/.neurovault/
  data/brain.db          # SQLite database (WAL mode)
  data/hnsw.bin          # HNSW vector index
  vault/                 # Markdown files (Obsidian-compatible)
  export/                # Auto-generated exports for AI assistants
  finetune/              # Training scripts and datasets
  backups/               # Automated backups
```

## The 31 Circuits

Circuits are NeuroVault's autonomous self-improvement engine. One circuit runs every 20 minutes, selected by a rotation dispatcher that avoids repeating any circuit within the last 3 cycles. This yields 72 improvement cycles per day.

## Phase Omega Systems Overview

### Digital Twin
Builds a cognitive model of the brain itself -- a fingerprint of strengths, preferences, and thinking patterns. The decision simulator uses this model to predict how the brain would approach novel problems.

### Agent Swarm
Five specialist agents (researcher, analyst, critic, synthesizer, planner) that can be dispatched to investigate topics autonomously. Goals are decomposed into tasks and assigned to the most suitable agent.

### World Model
Tracks external entities, causal relationships, and temporal patterns. Can run "what-if" simulations by propagating hypothetical changes through the causal graph.

### Recursive Self-Improvement
The brain compiles its knowledge into executable rules, optimizes its own circuits based on performance metrics, and tracks its expanding capabilities over time.

### Sensory Expansion
Extends beyond text: image analysis (via vision models), audio transcription, and live data streams (RSS feeds, APIs) feed new knowledge into the brain continuously.

### Federation
Multiple NeuroVault instances can form a collective, sharing knowledge selectively while maintaining local sovereignty. Each brain controls what it shares and what it accepts.

### Infrastructure
Distributed architecture support with edge caching, cluster health monitoring, and system resource tracking.

### Economic Autonomy
Tracks the tangible value the brain generates (time saved, decisions improved) against its compute costs (CPU, GPU, storage), calculating ROI and sustainability metrics.

### Consciousness Layer
The most speculative layer: a self-model that the brain maintains about itself, an attention mechanism that directs focus to the most promising areas, and a curiosity engine that drives autonomous exploration.
